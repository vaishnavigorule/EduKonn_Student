# 🎥 Test Execution Evidence

## 📖 Overview
This folder contains screen recordings captured during the manual testing of the **EduKonn Student App**. These recordings provide visual evidence of the test execution process and the defects identified during testing.

## 📂 Contents
- Screen recordings of executed test cases
- Recordings demonstrating identified defects
- Evidence supporting defect reports and test execution results

## 📌 Purpose
The recordings are included to:
- Validate executed test scenarios
- Demonstrate application behavior
- Provide evidence for reported defects
- Assist in defect verification and review
